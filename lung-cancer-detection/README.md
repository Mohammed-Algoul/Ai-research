# Lung Cancer Detection from CT Scan Images using Deep Learning

## Overview

This project presents a deep learning approach for detecting lung cancer from CT scan images using Convolutional Neural Networks (CNNs).

## Objective

The main goal of this project is to explore how CNN models can assist in the early detection of lung cancer from medical images.

## Technologies

* Python
* TensorFlow / Keras
* Convolutional Neural Networks (CNN)
* Medical Image Processing
